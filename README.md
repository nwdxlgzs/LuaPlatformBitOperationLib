# LuaPlatformBitOperationLib
一言以蔽之：试探平台整数上限，并以此动态实现位运算中如bnot反转定长的问题，不用依赖定长bit了（负数补码）
> 如果你需要压缩代码，我们已经帮你准备好了，直接拿去用吧，见带`-min`的文件即可。

# 支持的位运算
### 按位与
函数：`band`对应的指令`&`
### 按位或
函数：`bor`对应的指令`|`
### 按位异或
函数：`bxor`对应的指令`~`
### 按位反
函数：`bnot`对应的指令`~`
### 按位左移
函数：`lshift`对应的指令`<<`
### 按位右移
函数：`rshift`对应的指令`>>`

> 剩下未提供，自行解决，本库目的是解决Lua53之后定义了上述6个位运算指令在低版本Lua上等效依附平台(双精度浮点53位，整数32或者64位)实现位运算。
